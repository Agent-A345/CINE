## 🎬 CINE — Custom Video Player

A cinematic, feature-rich Custom Video Player Web App built using HTML, CSS, and JavaScript, offering advanced playback controls, playlist management, resume functionality, and immersive viewing experience.

## 🚀 Features
* **Custom Video Player UI** – Fully customized controls replacing default browser video player.
* **Playlist System** – Multiple videos with dynamic switching and active state tracking.
* **Resume Playback Feature** – Automatically resumes videos from last watched position.
* **Multi-Source Fallback System** – Switches video sources automatically on load failure.
* **Advanced Playback Controls** – Play, pause, rewind, forward, restart functionality.
* **Interactive Progress Bar** – Seek, hover preview, and draggable timeline control.
* **Keyboard Shortcuts Support** – Control playback using keyboard (Space, Arrows, F, M).
* **Playback Speed Control** – Adjust speed from 0.5× to 2× dynamically.
* **Volume & Mute Control** – Smooth volume slider with dynamic icon updates.
* **Picture-in-Picture Mode** – Watch videos in floating window mode.
* **Fullscreen Mode** – Toggle fullscreen viewing with smooth UI transitions.

## 🎮 How to Use
**Play & Control Video**
* Click video or Play button to start
* Press Space to toggle play/pause

**Navigate Timeline**
* Drag progress bar to seek
* Hover to preview timestamp
* Click to jump to position

**Change Playback Speed**
* Click speed button
* Select from 0.5× to 2×
